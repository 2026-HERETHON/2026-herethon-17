# 🪴 HERb
***HE:RE**, **‘여기’** 에서 다시 시작하다.*
<br>
> **HERb**는 'RE'를 여성의 생애 전환점, 즉 완경 이후의 새로운 시작으로 정의했습니다.

### ⭐️ 서비스 소개
> *완경이라는 전환점 앞에 선 여성들이 <br> 증상을 스스로 이해하고 다음 챕터를 다시 시작하도록 돕는 셀프케어 서비스*

<br>HERb는 완경기 여성들이 변화의 시기를 혼자 견디지 않고, 
<br>기록·연결·자기이해를 통해 다시 시작할 수 있도록 지원합니다.

완경을 종료나 상실이 아닌, 
<br>**다음 챕터를 준비하는 전환점**으로 재해석하는 것이 HERb 서비스의 출발점입니다.

이를 바탕으로 사용자가 자신의 변화를 기록하고, 
<br>공감과 자기이해를 통해 완경기를 자연스럽게 받아들이며, 
<br>**새로운 삶을 시작**할 수 있도록 돕습니다.

### 🌐 핵심 기능

- 🩺 자가진단 — 5문항 설문을 통해 현재 여정을 혼란기·적응기·재도약기 중 하나로 진단
- 📋 증상 트래커 — 5가지 주요 완경 증상을 약·중·강 3단계로 기록하여 일별 증상 관리
- 📈 패턴 리포트 — 주간·월간 증상 변화를 시각화하여 자신의 패턴을 한눈에 확인
- 🌱 여정 타임라인 — 자가진단 결과에 따른 단계별 콘텐츠와 여정을 제공
- 🌼 나의 정원 — 완경 이후 새롭게 시작한 도전과 일상을 기록하는 성장 공간
- 💬 허브 커뮤니티 — 익명으로 경험을 공유하고 공감과 정보를 나누는 커뮤니티
- 🏥 가까운 산부인과 찾기 — 지도 서비스와 연동하여 주변 산부인과 정보를 확인
- 🔔 알림 설정 — 증상 기록 리마인더와 알림 수신 여부를 설정

### 🚀 개발환경에서의 실행 방법
```bash
$ cd herb/
$ python -m venv myvenv
$ myvenv\Scripts\activate
$ pip install -r requirements.txt
$ python manage.py migrate
$ python manage.py runserver
```
